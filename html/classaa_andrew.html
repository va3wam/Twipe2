bool aaAndrew::start ( const char *  HOST_NAME)

Manage the process of connecting to the WiFi network.

Scan for a suitable Access Point and if one is found then connect to it using the init_wifi() method. This method connects to the Access Point, Starts up a webserver that serves up a web page that allows you to configure network settings withut recompiling as well as handling Over The Air updates using websockets.

@return bool where true means a conneciton was made and false means no connectioon was made.
